Для бухгалтерии ресторана рекомендуется создать несколько таблиц Excel, каждая из которых будет отслеживать различные аспекты финансовой деятельности. Вот структура таблиц, которые могут понадобиться:

1.  Таблица с доходами
	-   A: Дата
	-   B: Категория (например, продажи блюд, напитков, кейтеринг)
	-   C: Описание
	-   D: Сумма
	-   E: НДС (если применимо)

	Формула для подсчета итоговой суммы с НДС в столбце F: =D2*(1+E2)
---
2.  Таблица с расходами
	-   A: Дата
	-   B: Категория (например, закупка продуктов, зарплата, аренда)
	-   C: Описание
	-   D: Сумма
	-   E: НДС (если применимо)
	
	Формула для подсчета итоговой суммы с НДС в столбце F: =D2*(1+E2)
---
3.  Таблица с зарплатой сотрудников

	-   A: ФИО сотрудника
	-   B: Должность
	-   C: Стаж работы (в годах)
	-   D: Базовая ставка за час
	-   E: Коэффициент стажа
	-   F: Ставка за час с учетом стажа (D2 * E2)
	-   G: Оклад
	-   H: Количество отработанных часов
	-   I: Оплата почасовой работы (H2 * F2)
	-   J: Премии
	-   K: Начислено (оклад + оплата почасовой работы + премии)
	-   L: Налог на доходы физических лиц (НДФЛ, например, 13%)
	-   M: Сумма налога (K2 * L2)
	-   N: Пенсионные отчисления (например, 22%)
	-   O: Сумма пенсионных отчислений (K2 * N2)
	-   P: Зарплата к выдаче (начислено - сумма налога - сумма пенсионных отчислений)
	
	Формулы для расчета различных значений в этой таблице:
	
	-   Ставка за час с учетом стажа (F2): =D2*E2
	-   Оплата почасовой работы (I2): =H2*F2
	-   Начислено (K2): =G2+I2+J2
	-   Сумма налога (M2): =K2*L2
	-   Сумма пенсионных отчислений (O2): =K2*N2
	-   Зарплата к выдаче (P2): =K2-M2-O2
	
	Формула для подсчета оплаты почасовой работы в столбце H: =D2*E2 Формула для подсчета начисленной зарплаты в столбце G: =C2+H2+F2
---
4.  Таблица с анализом доходов и расходов
	-   A: Месяц
	-   B: Общий доход (ссылка на сумму столбца F из таблицы с доходами)
	-   C: Общий расход (ссылка на сумму столбца F из таблицы с расходами)
	-   D: Прибыль (разница между доходами и расходами)

	Формула для подсчета прибыли в столбце D: =B2-C2
---
5.  Таблица с инвентаризацией
	-   A: Наименование товара
	-   B: Категория (например, продукты, напитки, посуда)
	-   C: Единица измерения (например, кг, л, шт)
	-   D: Количество на начало периода
	-   E: Приход
	-   F: Расход
	-   G: Количество на конец периода (остаток)
	
	Формула для подсчета остатка на конец периода в столбце G: =D2+E2-F2
