3.  Добавь заголовок первого уровня (`<h1>`) и абзац (`<p>`) внутри тега `<body>`:


```html   
<h1>Добро пожаловать на мой сайт</h1>
<p>Здесь я буду делиться своими мыслями и идеями.</p>

```

4.  Создай список из трех элементов, используя теги `<ul>` (маркированный список) и `<li>` (элемент списка):

```html
<h2>Мои интересы</h2>     
<ul>
	<li>Веб-разработка</li>
	<li>Фотография</li>
	<li>Путешествия</li>
</ul> 
```

5.  Добавь гиперссылку на любимый ресурс, используя тег `<a>`:

```html
<h2>Полезная ссылка</h2>     
<p>
	<a href="https://www.example.com" target="_blank">Перейти на пример.com</a>
</p>
```
